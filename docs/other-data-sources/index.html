Other Data Sources

If your algorithm needs to read or write data files from a data source for which there is no Data Connector, you can connect directly to that provider from within your algorithm using an API or client library for that data source, while storing credentials outside of your algorithm source by using Hosted Data and the Algorithmia Data API.
